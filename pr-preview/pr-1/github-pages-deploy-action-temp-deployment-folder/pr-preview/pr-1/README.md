# Bride Mode - Bridal Landing Page

A beautiful, responsive landing page for a bridal business, built with HTML and CSS.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Elegant design with smooth animations and transitions
- **Key Sections**:
  - Navigation bar with smooth scrolling
  - Hero section with call-to-action
  - Services showcase with 4 service cards
  - About section
  - Contact form with business information
  - Footer with social links

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, animations)
- Google Fonts (Playfair Display, Montserrat)

## Getting Started

1. Open [index.html](index.html) in your web browser
2. Or use a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser

## Customization

### Colors
The main brand color is gold (`#d4af7a`). You can change this in [styles.css](styles.css) by searching for `#d4af7a` and replacing it with your preferred color.

### Content
- Update business information in [index.html](index.html)
- Replace placeholder text with your actual content
- Add your own images by replacing the image-placeholder div

### Fonts
Currently using:
- **Playfair Display** for headings
- **Montserrat** for body text

Change these in the Google Fonts link in the HTML head section.
